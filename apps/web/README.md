# Rverity Web Interface

This is the core frontend application for the Rverity protocol. It provides the visual surface for the Knowledge Graph and acts as the command center for data ingestion and recall.

## Architecture

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS + Framer Motion
- **3D Engine**: React Three Fiber
- **State**: TanStack Query (Server) + Zustand (Client)

## Development

Run the development server from the root of the monorepo:

```bash
npm run dev
```

For detailed documentation, please refer to the [root README](../../README.md).
